# Aubergine 🍆

A truly static, vanilla JS and HTML recipe organizer with offline storage and a responsive Material 3 inspired design.

## 🚀 Truly Static & Zero Build
This application is built for simplicity and portability:
- **No Build Required**: Runs directly from the root `index.html`.
- **Vanilla Power**: Pure JavaScript, HTML, and CSS (Tailwind via CDN).
- **Offline First**: Uses **Dexie.js** (IndexedDB) to store all your recipes locally in your browser.

## 📱 Features
- **Responsive Navigation**: Material 3 Rail on desktop and a convenient Bottom Navigation on mobile.
- **Dedicated Search**: A fast, client-side search page that scans titles, descriptions, and ingredients.
- **Recipe Management**: Add, edit, favorite, and delete recipes with ease.
- **Image Support**: Attach photos to your recipes and view them in a polished detail view.

## 🛠 Tech Stack
- **HTML5 & CSS3**
- **Tailwind CSS** (CDN)
- **Dexie.js** (IndexedDB wrapper)
- **Lucide Icons** (CDN)
- **Vanilla JavaScript** (ES6+)

## 📂 Deployment
Deploying to GitHub Pages is as simple as uploading the files to your repository. It will work automatically at any sub-path (e.g., `username.github.io/repo-name/`).
